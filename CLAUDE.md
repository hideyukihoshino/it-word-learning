# IT用語ノートアプリ（ARI用語集）

## プロジェクト概要
ARIに入社後に出会ったIT・ビジネス用語をまとめた個人学習ツール。
スマホ（iPhone）でいつでも確認・追加できるWebアプリ。

## 技術スタック
- **フロントエンド**: Next.js（Vercel にデプロイ）
- **データベース**: Firebase Firestore
- **AI補助**: Claude API（説明の検証・補足）
- **ホスティング**: Vercel

## 機能
- 用語検索・カテゴリフィルター
- 用語追加・編集（簡単フォーム）
- 「AIに確認する」ボタンでClaudeが説明を検証・補足
- スマホ対応UI

## カテゴリ
1. ビジネス・プロジェクト運営
2. 営業・マーケティング
3. 契約・法務
4. IT全般・インフラ
5. クラウド（AWS / GCP）
6. 開発・アプリケーション基盤
7. データ分析・BI・データベース
8. AI・生成AI関連
